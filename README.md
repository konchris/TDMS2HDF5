TDMS2HDF5
=========

View National Instruments' TDMS files as well as HDF5 files and export them to
HDF5 or CSV.

Installation
============
`pip install -e git+https://github.com/konchris/TDMS2HDF5.git#egg=TDMS2HDF5`

#Usage

Usage is fairly easy.

### *Nix 

##### Bash
```bash
$ tdms2hdf5
```

### Windows

##### PowerShell
```powershell
PS C:\> tdms2hdf5.exe
```
